***  ВНИМАНИЕ!   Пакет   по  умолчанию  настроен  на  запуск
sshguard, установленный в /use/local!

Данный  набор скриптов предназначен для установки и удаления
сервиса SMF для sshguard daemon на Sun Solaris 10.

Для активизации sshguard SMF выполните следующие шаги:
------------------------------------------------------

1.  Установите sshguard, если это еще не сделано.

2. Сконфигурируйте sshguard.

3.  При  необходимости, отредактируйте параметр (переменную)
BASE_DIR   в  скрипте  init.sshguard, чтобы  она  указывала
на каталог, в котором реально установлен sshguard.

4.    Запустите    скрипт   sshguard_smf_inst.sh,   который
и    выполнит    все     необходимые     пререквизиты      и
постинсталляционные   действия  и  установит  sshguard  SMF
сервис.

5.    Выполните   команду   svcadm   enable   sshguard  для
активизации и старта сервиса sshguard.

Для деактивации и удаления sshguard SMF сервиса выполните
---------------------------------------------------------
следующие шаги:
---------------

1.Запустите  скрипт sshguard_smf_rmv.sh  . Скрипт останавливает
все  работающие  процессы sshguard, рарзегистрирует SMF сервис,
полностью    удаляет   sshguard  SMF   и  откатывает  операции,
выполненные ранее при активизации сервиса.

***  Замечание:   Удаление  сервиса sshguard  SMF не приводит к
удалению установленного ПО SSHGuard.

Архив содержит:

init.sshguard           - Управляющий метод sshguard SMF
sshguard.xml            - Сервисный манифест sshguard SMF
readme_ru.txt           - Этот файл (Russian)
readme_en.txt           - Этот файл (English)
sshguard_smf_inst.sh    - Установочный скрипт SSHGuard SMF
sshguard_smf_rmv.sh     - Скрипт удаления SSHGuard SMF
